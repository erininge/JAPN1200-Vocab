# Kat’s Class Vocab — JPLN1200 (PWA)

Version: V3.4

## Audio
Put audio files in `/audio` named exactly `<id>.<ext>`.
Supported extensions: `.wav`, `.mp3`, `.m4a`, `.ogg`.

Example:
- `audio/jpln1200_l1_001.wav`

## Add vocab / lessons
Edit or add JSON files in `/lessons`.
Then update `/lessons/index.json` with the lesson metadata (or ask Codex).
